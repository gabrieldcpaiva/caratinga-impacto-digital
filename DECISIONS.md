# DECISIONS: Local Business Impact v2 (Hub)

- **Centralização:** Decidimos unificar os 5 novos negócios com os 5 anteriores no GitHub Pages para criar uma vitrine de impacto única.
- **Design:** Aplicamos o "The Void Principle" (Estética Cinematográfica) nos 5 novos negócios, utilizando modo escuro, tipografia refinada e espaçamento generoso para transmitir luxo e exclusividade.
- **Estrutura de Arquivos:** Cada negócio possui sua pasta com `index.html` (Landing Page Completa) e `proposta.html` (Proposta), facilitando a navegação relativa.
- **Hub Upgrade (Henry):** Refizemos a index.html aplicando o pacote de design UI Designer (Minimalist Luxury), com tipografia 'Outfit'/'Inter', cores dark e glassmorphism para elevar a percepção de valor.
- **Outreach Estratégico:** Decidimos não usar abordagens genéricas. O outreach gerado para cada cliente ataca diretamente o problema de "demanda reprimida invisível" e oferece o hub local como solução.
